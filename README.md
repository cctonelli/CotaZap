O **CotaZap** transcende o conceito de um simples app de cotação. Ele é uma plataforma de **Inteligência de Suprimentos** desenhada para aniquilar a ineficiência no processo de compras via WhatsApp. Transformamos conversas informais em workflows estruturados e decisões baseadas em dados.

---

### 🚀 O Poder do Comprador (Destaques da Versão)

O CotaZap dá ao Comprador superpoderes de negociação com ferramentas de última geração:

*   **📲 Disparo em Massa via WhatsApp**: Envie cotações detalhadas para dezenas de fornecedores simultaneamente com um único clique, utilizando a potência da **Evolution API v2**.
*   **📥 Recepção e Centralização de Ofertas**: Chega de planilhas manuais. Receba múltiplas respostas e deixe o CotaZap organizar tudo para você.
*   **📊 Comparador Inteligente de Preços**: Visualize e compare todas as ofertas recebidas em uma interface intuitiva. O sistema identifica automaticamente as melhores condições para que você economize de verdade.
*   **💎 Interface Premium de Elite**: Design moderno com **Material 3**, suporte a **Glassmorphism** e **Dark Mode**, inspirado no que há de mais avançado em UI/UX.

---

### 🛠️ Stack Tecnológica

| Camada | Tecnologia | Propósito |
| :--- | :--- | :--- |
| **Mobile** | Flutter (Dart) | UI Premium de Elite e Performance Nativa |
| **Local DB** | Drift (SQLite) | Persistência ultra-rápida e operação Offline-First |
| **Cloud/Auth** | Supabase | Gestão de Usuários, Roles e Sincronização |
| **Mensageria** | Evolution API v2 | Motor de WhatsApp profissional e escalável |
| **Backend** | FastAPI (Python) | Orquestração de webhooks e lógica de negócios |

### 🎯 AS MISSÕES: O Futuro do Procurement

*   **🟢 MISSÃO 1: Inteligência Artificial (AI-OCR)**: Leitura automática de tabelas de preços em fotos e PDFs para eliminar o registro manual.
*   **🔵 MISSÃO 2: Rede CotaZap (Marketplace)**: Descoberta inteligente de novos fornecedores por categoria e região.
*   **🟡 MISSÃO 3: Analytics e Tendências**: Gráficos de variação de preços e sugestão de melhor momento para compra.
